# 前言

SSH 登录服务器，每次都需要输入密码，流程繁琐。

特别是在使用 remote-ssh （VScode 插件，集成可视化和终端）时，每次将服务器内的文件夹添加到 VScode 添加到服务器时，都需要手动输入密码。

因此使用 SSH 免密登录简化流程是必备的步骤。

# 教程

  1. 查看本地密钥
# 如果拥有本地密钥，本地会拥有.ssh文件夹用于存放密钥
ls -la ~/.ssh
  • 根据输出内容查看是否含有本地密钥相关文件

::: theorem 相关问题

ls -la 的作用及输入内容含义，可以查看该文章：一份前端够用的 Linux 命令 (opens new window)

还没拥有本地密钥 ❓ 生成本地密钥（SSH key）：本地终端输入命令 ssh-keygen -t rea

::: right

ssh-keygen 作用及相关参数：阅读此文章 (opens new window)

:::

  1. 本地密钥确认已经拥有后，可以执行命令 ssh-copy-id -i ~/.ssh/id_rsa.pub root@43.138.158.103 ，该命令可以将公钥文件传输到服务器上，并生效：
ssh-copy-id -i ~/.ssh/id_rsa.pub root@服务器IP

这里服务器里的 authorized_keys中已经保存有本地公钥了，因此以后连接服务器将不再需要密码了，到此免密登录完成
